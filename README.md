# FeHelper 插件
FeHelper Plugins

## 插件

### 回声工具
这个工具的功能是：
1. 输入什么，就会输出什么；
2. 可以用于测试，或者调试；

### 随机工具

Prompt:

```shell
这是一个随机的工具，功能有：
1. 指定一个数字范围，生成一个随机数；
2. 生成一个随机字符串，可包含大小写字母、数字和特殊字符，指定生成的长度；
3. 生成一个随机密码，可包含大小写字母数字，和少量字符；
4. 生成一个UUID的字符串；

目前就这些功能，后期可能会增加，想到了就会增加功能；
请来在这里面实现当前的功能，页面布局和排版要合理，美观；你是一个会UI的前端工程师；只能使用原生的JS和CSS来；不能依赖外部的包和引用。
```
